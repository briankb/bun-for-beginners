# Bun for Beginners

Code for the book **[Bun for Beginners](https://learnbun.org)** — a project-based introduction to the Bun JavaScript/TypeScript toolkit.

## Requirements

- [Bun](https://bun.com) 1.0 or later
- Git

Chapter 1 walks through installing both.

## About the book

Bun for Beginners takes you from an empty folder to a deployed app. No prior Bun experience needed. JavaScript or TypeScript familiarity helps but isn't required.

Read it at **[learnbun.org](https://learnbun.org)**.
